# Goal Tracker MERN 🎯

A full-stack web application built on the **MERN stack** (MongoDB, Express, React, Node.js) to help users set, track, and achieve their personal and professional goals.

## 🌟 Features

- **Goal Management**: Create, Read, Update, and Delete (CRUD) goals with titles, descriptions, and deadlines.
- **Categorization & Priority**: Organize goals into categories (Study, Fitness, Career, Finance, Personal) and set priority levels (Low, Medium, High).
- **Progress Tracking**: Update progress percentages and status (Pending, In Progress, Completed).
- **Responsive Dashboard**: A modern React-based frontend with a clean dashboard to visualize all active goals.
- **Backend API**: A robust Node.js/Express backend with Mongoose for data persistence.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS, Framer Motion, Recharts
- **Backend**: Node.js, Express
- **Database**: MongoDB (via Mongoose)
- **State Management**: React Hooks & Context API

## 🚀 Setup Instructions

### Backend
1. Navigate to `server/`:
   ```bash
   cd goal-tracker/server
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set your `MONGO_URI` in environment variables.
4. Start the server:
   ```bash
   npm start
   ```

### Frontend
1. Navigate to `goal-tracker/`:
   ```bash
   cd goal-tracker
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
